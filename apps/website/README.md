# Flooring Website - SEO & Conversion Optimized

Website profissional para empresa de Flooring (pisos e revestimentos) nos Estados Unidos, otimizado para SEO local e geração de leads.

## ?? Objetivos

- Dominar buscas locais ("flooring installer near me", "hardwood flooring installation + city")
- Maximizar conversão de leads (calls, forms, WhatsApp)
- Core Web Vitals excelentes
- Arquitetura escalável para múltiplas cidades e serviços

## ??? Estrutura

- **Homepage**: Hero com CTA, prova social, serviços
- **Páginas de Serviços**: 8 páginas individuais (Hardwood, Vinyl, Tile, Epoxy, Refinishing, Repairs, Commercial, Residential)
- **Landing Pages Locais**: Estrutura programática para SEO local
- **Páginas de Conversão**: Free Estimate, Schedule Measurement, Contact
- **Blog**: SEO de conteúdo com internal linking

## ?? Tecnologias

- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- React Hook Form + Zod
- SEO técnico completo

## ?? Instalação

```bash
npm install
npm run dev
```

## ?? SEO Features

- Schema markup (LocalBusiness, Service, FAQ, Review)
- Sitemap.xml automático
- Robots.txt otimizado
- Meta tags dinâmicos
- URLs amigáveis
- Breadcrumbs
- Lazy loading de imagens

## ?? Performance

- Lighthouse score > 90
- Core Web Vitals otimizados
- SSR/SSG para SEO
- Imagens otimizadas (AVIF/WebP)
